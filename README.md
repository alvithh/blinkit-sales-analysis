# Blinkit Sales Dashboard

## Project Overview
This project analyzes Blinkit sales data using Power BI to uncover trends, outlet performance, and customer behavior.

## Tools Used
- Power BI
- Excel
- SQL (for preprocessing)

## Key Insights
- Total sales: $1.2M
- Performance comparison by outlet size and location
- Top-selling categories identified

## Files
- BlinkitDashboard.pbix – Power BI dashboard
- Dataset.xlsx – Raw data
